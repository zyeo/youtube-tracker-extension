# YouTube Time Tracker Chrome Extension

A simple Chrome extension for tracking YouTube usage across videos, Shorts, and browsing.

## Features

- Tracks total time spent on YouTube
- Breaks down usage by videos, Shorts, and browsing
- Displays daily usage metrics in a lightweight popup
- Shows trends and changes in usage over time

## Tech Stack

- JavaScript
- Chrome Extension APIs
- HTML / CSS

## How It Works

- Listens for activity on YouTube pages
- Categorizes usage (video, Shorts, browsing)
- Stores time data locally
- Displays insights through a popup dashboard

## Installation

1. Clone this repository
2. Go to `chrome://extensions/`
3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select this project folder
